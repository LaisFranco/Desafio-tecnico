# Este repositório contém a solução de um desafio técnico composto por dois casos independentes, com foco em análise de problema, proposta de solução e estratégia de testes.

## Visão Geral dos Desafios

**SQL – Permissões RBAC**

- Análise e correção de um problema de permissões de usuário
- Solução baseada em modelagem correta de papéis
- Implementação completa em SQL

**API – Resources (Open Finance Brasil v3.0.0)**

- Análise do contrato e changelog da API
- Criação de um Plano de Testes completo
- Definição de estratégia de automação com Postman + Newman
- Checklist executável derivado do plano
- Scripts reutilizáveis de pré-request e validação

**As URLs da documentação oficial são apenas exemplos (placeholders).
Por isso, os testes não foram executados, apenas analisados e planejados.**
